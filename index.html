Introduction

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.

CSS Syntax

CSS is a rule-based language â€” you define the rules by specifying groups of styles that should be applied to particular elements or groups of elements on your web page.

h1 { color: red; font-size: 12px; }
  • The CSS rule opens with a selector. This selects the HTML element that we are going to style. In this case, we are styling level one headings (h1).
  • Inside the curly braces, there are declarations specifying the property and value pairs (e.g., color and font-size).
  • The example contains two declarations, one for color and the other for font-size.
  • CSS properties can take different values depending on the property (e.g., color can be any valid color name or code).
How CSS Structure Works

Let's examine the structure of CSS. CSS can be applied to a document in three ways: external stylesheet, internal stylesheet, and inline styles.

External Stylesheet

An external stylesheet is the most common method of bringing CSS to a document. You can link a single CSS file to multiple web pages, styling all of them with the same CSS stylesheet.

<link rel="stylesheet" href="styles.css">

Internal Stylesheet

An internal stylesheet resides within an HTML document, within the <style> element inside the <head> section.

<style> h1 { color: blue; background-color: yellow; } </style>

Inline Styles

Inline styles are applied directly to individual HTML elements using the style attribute.

<h1 style="color: blue; background-color: yellow;">Hello World!</h1>
How Does CSS Actually Work

When a browser displays a document, it must combine the document's content with its style information. It processes the document in several stages.

  1. The browser loads the HTML document.
  2. The browser converts the HTML into a Document Object Model (DOM), representing the document in memory.
  3. The browser fetches resources like images, videos, and linked CSS files.
  4. The browser parses the fetched CSS and applies the rules to the DOM elements, creating a render tree.
  5. The render tree is painted to the screen, displaying the page.
What is a Selector

A CSS selector specifies which HTML elements should be styled. It can target elements by their tag name, class, or ID.

h1 { color: blue; } .special { color: blue; }
Types of Selectors

Selectors allow you to target specific HTML elements. There are various types of selectors:

Attribute Selectors

These selectors target elements based on their attributes.

Pseudo-classes and Pseudo-elements

Examples include :hover for hovering over an element, and ::first-line for targeting the first line of an element.

Combinators

Combinators allow you to select elements based on their relationships with other elements, like > for selecting child elements.

article > p {}
